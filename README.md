# Only-Train-Once
This repository contains the official implementation of the paper: "Only Train Once: Uncertainty-Aware One-Class Learning for Face Authenticity Detection", currently under submission to TMM. The complete source code, pre-trained models, and configuration files will be made publicly available upon the acceptance of the paper.
